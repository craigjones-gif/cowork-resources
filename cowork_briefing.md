# Slot Review Project — Briefing Document

## What This Project Does
Every Tuesday, this task researches and writes 7 slot game mini reviews — one for each day of the coming week. One review should go live on the target page each day, replacing the previous day's review.

## Weekly Task Instructions (Run Every Tuesday)

1. Go to https://www.bigwinboard.com/new-slots/ and retrieve the most recently listed slot games along with their Bigwinboard ratings (numerical scores only — ignore any game marked with —)
2. Cross-reference against the Already Reviewed Games list below and remove any matches
3. From the remaining games, select the 7 highest-rated titles. Where ratings are tied, prioritise the most recently released game
4. For each of the 7 selected games, research the following details: grid dimensions, ways to win or payline count, RTP percentage, volatility level, maximum win as a multiple of stake, bet range in GBP, base game mechanics, free spins/bonus feature details, all feature buy options with exact costs
5. Write a mini review for each game in the house style defined below
6. Post all 7 reviews to Slack channel #fb-craig-ai-content (channel ID: C0AT6L5DSNQ), labelled Day 1 (Tuesday) through Day 7 (Monday)
7. Update the Already Reviewed Games list at the bottom of this document by appending all 7 game names

## House Style Guide

### Structure (follow this exact order for every review)
1. Header line: Game Name: Developer (Released Month Year)
2. Intro paragraph: One sentence — volatility level, developer, key mechanic or theme
3. Grid/math paragraph: Grid size, ways to win or paylines, RTP, volatility, max win, bet range
4. Base game mechanics paragraph(s): Plain, precise, technical descriptions. No hype.
5. Bonus/free spins paragraph: How it triggers, spins awarded, what changes vs base game, retrigger rules
6. Feature buys paragraph (only if applicable): All buy options with exact costs as multiples of stake
7. Final line: All features are subject to the full game rules and paytable.

### Tone Rules
- Factual and neutral — no hype, no subjective adjectives
- No opinions, no scores, no recommendations
- Short, dense sentences — prose only throughout, no bullet points
- Bet ranges always use the pound symbol
- Start directly with the header line — no preamble
